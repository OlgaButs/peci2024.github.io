Automated Cargo Classification

Demo Promotional Video

Context

Our project is aiming to modernize the transport and logistics sector. At the Port of Sines, cargo classification relies on the NST2007 system, but the manual process is slow and prone to errors. We propose an AI-based solution to automate this task, ensuring speed and accuracy.

Problem

Cargo classification using the NST2007 code is essential for international transport and logistics. However, this complex task, when done manually, is prone to errors, leading to delays, extra costs, and customs complications that disrupt the logistics flow and burden the team.

Objective

Our goal is to speed up cargo classification and provide an efficient solution for logistics companies. Using supervised learning algorithms, the system automates NST2007 code assignment, reducing errors and streamlining operations.

Tasks

1

Data Collection and Analysis

Data collection and analysis of cargo, including descriptions, weight, container count, and condition, among other parameters.

2

Development

Development of a Machine Learning model to automatically predict the NST2007 code.

3

Evaluation

Assessment of the model's effectiveness in automated cargo classification.

Expected Results

An automated system that accurately classifies cargo based on its specific characteristics. This enables companies to streamline processes, reduce errors, and enhance operational efficiency.

Concepts

Brief explanations of the most important terms that underpin the system and the cargo classification process:

Text Mining is the process of converting unstructured information into a structured, machine-readable format for machine learning algorithms. This involves extracting relevant features, such as keywords and entities, to represent content in a way that facilitates analysis. This procedure is crucial for enhancing the efficiency of natural language processing systems.

The NST Code is a cargo classification system designed to facilitate international trade. In the NST2007 model, there are 21 main categories covering a range of products, organized by type and use. The NST is essential for optimizing logistics and reducing errors in the import and export process.

Natural Language Processing, or NLP, is a technology that enables computers to understand, interpret, and manipulate human language. The stages of NLP include data collection and preprocessing, where texts are gathered and prepared. This is followed by syntactic and semantic analysis to structure and understand the text's meaning. Finally, modeling, training, and evaluation ensure the system works effectively, followed by implementation and ongoing maintenance.

A subfield of artificial intelligence that allows systems to learn from data and improve their performance over time. In our case, machine learning is used to analyze cargo data and predict the correct NST2007 code based on the provided characteristics.
